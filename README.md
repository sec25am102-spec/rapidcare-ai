# 🚑 RapidCare AI

AI-powered Emergency Response & Crisis Coordination System for Hospitals.

## 🔥 Problem
Hospitals lose precious minutes during emergencies due to slow manual communication.

## 💡 Solution
RapidCare AI detects emergencies and alerts doctors instantly using Google Cloud AI.

## ✨ Features
- AI Emergency Severity Prediction
- Real-time Alerts to Doctors & Nurses
- Live Hospital Dashboard
- Voice-based Emergency Reporting
- Ambulance Auto Dispatch

## 🛠 Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Firebase  
AI: Google Vertex AI  
Cloud: Google Cloud Platform
